## copyDirectory
a function of copy file inside directory use node

## Install 
npm
```bash
  $ npm intall copydirectory --save
```

yarn 
```bash 
  $ yarn add copydirectory --save
```

## Usage
copydirectory(src, dest)
  * src 源目录
  * dest 目标目录

```js
  var copydirectory = require('copydirectory');
  copydirectory('./dist', './build-zip')
```
